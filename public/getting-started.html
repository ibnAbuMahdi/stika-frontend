Getting Started Guide

Welcome to Stika! Transform your advertising strategy with Nigeria's premier tricycle advertising platform. This guide will help you create your first successful campaign in minutes.

1Complete Your Agency Setup

1.1 Verify Your Email

  • Check your inbox for the verification email
  • Click the verification link to activate your account
  • If not received, check spam folder or request a new verification email

1.2 Complete Agency Profile

Navigate to Settings > Profile and provide:

  • Business Information: Agency name, type, registration number
  • Contact Details: Business address, phone, website
  • Tax Information: Tax ID and business registration details
  • Agency Settings: Logo upload, brand colors, subscription tier

1.3 Account Verification

  • Upload business registration documents
  • Provide tax identification certificates
  • Wait for admin approval (typically 24-48 hours)
  • Verified agencies get access to all premium features

2Fund Your Wallet

2.1 Bank Transfer

  • Go to Wallet > Top-up
  • Transfer the amount you wish to top up into the displayed bank account.
  • Send the evidence of payment to support@stika.ng. Your wallet will be topped up with the confirmed amount.

2.2 Online Payment

  • Navigate to Wallet > Top-up
  • Select Pay with Monnify
  • Follow the next steps to complete payment.
  • Funds appear in wallet within minutes

3Connect to Your First Client (Optional)

3.1 Generate Invite Code

  • Go to Clients > Invite Codes
  • Create new invite code with custom name
  • Set usage limits and expiry date
  • Share code with client for easy onboarding

3.2 Respond to Client Requests

  • Go to Clients
  • View list of Client Join Requests
  • View a Requests > Accept/Decline Request.
  • Start creating campaign for Client

4Create Your First Campaign

4.1 Campaign Basics

  • Click "Create Campaign" from dashboard
  • Choose campaign template or start from scratch
  • Enter campaign name and description
  • Select client

4.2 Campaign Details

  • Upload advertisement images/stickers
  • Set optional details like instructions and campaign verifications frequency

4.3 Set Target Locations

  • Use interactive map to draw geofences
  • Target high-traffic areas like markets, schools, transport hubs
  • Each geofence should be at least 5000 meters in radius for optimal results
  • Set geofence duration, budget, rate and rider slots

4.4 Sticker Pickup Locations

  • Set sticker pickup locations for each geofence
  • Ensure that location details are correct

4.5 Campaign Funding

  • Select funding source (agency, client or agency and client)
  • Ensure that funds are available for the selected source

4.6 Review and Launch

  • Review all campaign settings
  • Check wallet balance is sufficient
  • Save as Draft or Submit for approval. Draft campaigns can be edited later and will not be reviewed for approval until submitted.

5Post Campaign Creation

5.1 Draft Campaign

  • Edit draft campaign by clicking the edit icon on it's card in the campaigns page.
  • Delete it by clicking the basket icon on it's card

5.2 Submitted Campaign

  • Expect feedback from the review team within 24 hours after submission.
  • If approved, the campaign can be launched immediately. If rejected, it can be edited and resubmitted.

5.3 Active Campaign

  • Active campaign can be monitored in real-time
  • View live dashboard with active riders
  • Monitor geofence coverage and time spent
  • Review verification photos from riders
  • Download detailed performance reports

5.4 Completed Campaign

  • Review campaign performance and results
  • Download final reports and insights
  • Archive completed campaigns if needed

6Monitor and Optimize

5.1 Track Performance

  • View live dashboard with active riders
  • Monitor geofence coverage and time spent
  • Review verification photos from riders
  • Download detailed performance reports

Pro Tips for Success

  • Start Small: Begin with 2-3 geofences to understand performance
  • Rate Types: per_hour = fewer riders + more insights, fixed_daily = more riders + less insights
  • Target High-traffic Areas: Set higher budgets for geofences around high-traffic areas
  • Monitor Daily: Check dashboard daily for optimization opportunities
  • Seasonal Planning: Plan campaigns around events, holidays, and seasons
